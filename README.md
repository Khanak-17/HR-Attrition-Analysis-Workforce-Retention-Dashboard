📊 HR Attrition Analysis & Workforce Retention Dashboard

Python · SQL · Power BI · Excel

This project presents an interactive HR Analytics dashboard developed using Power BI to analyze employee attrition, workforce demographics, employee satisfaction, salary distribution, and retention trends.

The dashboard helps HR professionals and business managers identify key attrition factors, monitor workforce KPIs, improve employee retention strategies, and make data-driven business decisions through interactive visualizations and dynamic filtering.

📸 Dashboard Preview

Interactive Power BI dashboard showing employee attrition trends, workforce demographics, salary insights, department analysis, and employee retention metrics.
